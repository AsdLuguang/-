## 环境搭建：

+ 系统：``ubuntu 16.04``
+ 安装软件：``texlive-full``, ``texlive-xetex``

```bash
    sudo apt install texlive-full
    sudo apt install texlive-xetex
```
+ 编译：
```bash
    make
```

+ latex可视化工具：``texmaker``
```bash
    sudo apt install texmaker
```
